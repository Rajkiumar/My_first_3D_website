3d animation design for website

This is my first attempt at creating a 3D animation for web use. I designed a simple 3D object using Skechfab. The animation was integrated into a webpage with Three.js. It rotates and responds to user interaction smoothly. The goal was to test basic 3D workflows for web design. I focused on performance and clean visual presentation. This helped me understand 3D modeling and web rendering basics. Tools used: Blender, Three.js, HTML/CSS. Next, I plan to add lighting, shadows, and interactive features.
